## Как участвовать

1. Прочитайте Конституцию.  
2. Получите NFT-паспорт (минт на Polygon → инструкция в citizenship.md).  
3. Внесите предложение через форму в Discord или напрямую в Snapshot.  
4. Изменения в документы возможны **только** после одобрения DAO (≥65% при кворуме ≥20%).

Любые правки вне DAO-голосования считаются недействительными.

## Независимость

- Документы pinned на IPFS (CID в каждом файле).  
- История изменений — в git.  
- Экономика и гражданство — на блокчейне (не зависят от GitHub или иного информационого рессурса).
