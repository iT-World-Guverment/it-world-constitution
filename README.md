# iT-World Governance

**Автономный цифровой мир**  
Закрытая экономика · Децентрализованное управление · Полная независимость от платформ

## О проекте

iT-World — это самодостаточный цифровой мир, функционирующий как децентрализованная автономная организация (DAO).  
Он не претендует на физический суверенитет и не зависит от какой-либо централизованной платформы (Discord, Telegram, GitHub и др.).

Все ключевые правила, идентичность участников и экономика фиксируются на блокчейне и в децентрализованном хранилище (IPFS/Arweave), что обеспечивает устойчивость даже при потере временных каналов связи.

## Статус проекта (на 13 января 2026)

- **Genesis-версия** — v1.0  
- **Дата основания** — 13 января 2026  
- **Текущий статус** — зафиксированы учредительные документы (genesis)  
- **Голосование** — через Snapshot (пространство будет создано после запуска)

- **Временный хаб общения** — Discord-сервер iT-World  
- **Постоянная идентичность** — ENS-домен / IPFS CID genesis-документов (будут добавлены после фиксации)

## Как присоединиться

1. Ознакомьтесь с **[Конституцией](/docs/constitution.md)** и **[Декларацией Автономии](/docs/declaration.md)**  
2. Подключите кошелёк на Polygon (MetaMask, Rabby и др.)  
3. Получите NFT-паспорт гражданина (инструкция → [citizenship.md](/docs/citizenship.md))  
4. Подайте заявку на верификацию в Discord (канал #верификация-и-вступление)  
5. Выберите подразделение/министерство (по желанию)  
6. Начните участвовать в голосованиях и экономике

## Как вносить изменения

1. Прочитайте **[Процедурные правила](/docs/procedures.md)**  
2. Создайте предложение в Snapshot (после запуска пространства)  
3. Пройдите обсуждение и голосование согласно типу предложения  
4. После одобрения DAO → откройте Pull Request в этот репозиторий  
   (в описании PR обязательно указать ссылку на Snapshot-предложение)

**Прямые коммиты в main запрещены** (branch protection будет включена после genesis-коммита).

## Независимость и устойчивость

- Документы pinned на IPFS (CID указан в шапке каждого файла)  
- История изменений — в git (публичный репозиторий)  
- Гражданство — через NFT на Polygon (не зависит от GitHub)  
- Экономика и голосования — на блокчейне (Polygon)  
- Временные каналы связи (Discord, Telegram) — только для координации, не являются фундаментом проекта
